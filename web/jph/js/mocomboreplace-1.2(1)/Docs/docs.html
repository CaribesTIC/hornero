moCombo.replace Documentation

Never Wanted to style a select element? Just to discover you couldn't change the color of the selected item?

That's why moCombo.replace it's here

What's moCombo.replace?

moCombo.replace converts any select into a fully styleable (via css) ordered list.
And in the case of the single select element, it even allow you to change the arrow of the drop down menu.

Does it keep events added to the original select?

This version of moCombo, keeps onchange events (even if they where assigned using mootools, with the addEven call).

Does it keep options groups?

Of course it does. thay are converted to an unordered list.

So what you need?

moCombo.replace need mootools 1.3, it won't work in any previous version.

Which browsers is it compatible with?

It was tested with IE7, IE8, FF2, FF3, Safari 5, Chrome 2 and Opera 11. All under windows.

How to use it?

just include the mootools 1.3 javascript file, and the moCombo javascript file, then just add this call to your domready. 

	new moCombo.replace($$('select.replaceme'));

You can pass just 1 Elements or as many as you want (using css match).

Are there any extra included?

moCombo it's included, moCombo let's you create a new select Element out of nothing. Still, I think it's far more easy to use moCombo.replace
